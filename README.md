# PROJ-SwiftBar

A collection of my custom scripts for [SwiftBar](https://github.com/swiftbar/SwiftBar). This project serves as my personal dashboard for productivity and inspiration.

## 🚀 SwiftBar Setup
SwiftBar allows for infinitely customizable menu bar items.
* **GitHub Repository:** [https://github.com/swiftbar/SwiftBar](https://github.com/swiftbar/SwiftBar)

## 🛠 How to Build Your First Script

### 1. Configuration Path
I keep all my scripts in: `/Users/XX/System-Config-Macbook/`

### 2. Creating a Script
SwiftBar scripts follow the naming convention: `name.refresh_interval.extension`. 
* Example: `slogan.1d.sh` (Refresh once every day).

**The Template:**
Create a `.sh` file and add the following content:
